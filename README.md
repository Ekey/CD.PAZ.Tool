# CD.Unpacker :see_no_evil:
Tool for extract PAZ archives from game - [Crimson Desert](https://store.steampowered.com/app/3321460/Crimson_Desert)

### Usage
> CD.Unpacker "**INPUT_PAMT_FILE**"

> CD.Unpacker "**INPUT_PAMT_FILE**" "**UNPACK_DIRECTORY (OPTIONAL)**"

### Examples
> CD.Unpacker "**E:\Games\Crimson Desert\0008\0.pamt**"

> CD.Unpacker "**E:\Games\Crimson Desert\0008\0.pamt**" "**D:\CD_Unpacked**"

# Requirements / Build
* Microsoft .NET Framework 4.7.2
* Visual Studio 2019 / 2022
